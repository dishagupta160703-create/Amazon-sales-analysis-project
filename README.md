## Amazon Sales Data Analysis 📊


## Project Overview  :

*  This project focuses on analyzing Amazon sales data to understand sales performance, product demand, and profit trends. The goal of this analysis is to extract useful business insights that can help improve decision-making.

*  The project uses Excel for data cleaning, SQL for data analysis, and Power BI for creating interactive dashboards.


  
##  Objectives  :

*  Analyze sales performance across different product categories

*  Identify top-selling products

*  Understand monthly sales trends

*  Analyse order distriution across payment mode

*  Analyse review  count across product categories

*  Identify product selling by region

  

##  Tools & Technologies :

*  Microsoft Excel – Data cleaning and preprocessing

*  SQL – Data querying and analysis

*  Power BI – Dashboard and data visualization

  

##  Dataset Information  :

The dataset contains the following fields:

*  Order ID

*  Order Date

*  Product Id

*  Product Category

*  Price

*  Quantity

*  Discount Percent

*  Region

*  Payment Mode

*  Rating

*  Review count

*  Total Revenue

  

##  Data Analysis Process  :

*  Data Cleaning using Excel

*  Data Analysis using SQL queries

*  Data Visualization using Power BI



##  Key Insights  :

*  Certain product categories generate higher sales.

*  Sales trends vary across different months.

*  Some regions contribute more to overall revenue.
 

##  Dashboard  :

The Power BI dashboard provides visual insights including:

*  Sales by Category

*  Monthly Sales Trend

*  Distribution of orders by Payment mode

*  Top 5 Selling Products

*  Quantity sold by region

*  reviw count by product categories
